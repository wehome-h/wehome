# 第四节 循环语句

理解：循环语句具有在某些条件满足的情况下，反复执行特定代码的功能。

<div class="br"></div>

循环结构分类：

  - for 循环

  - while 循环

  - do-while 循环

<div class="br"></div>

循环结构四要素：

  - 初始化部分

  - 循环条件部分

  - 循环体部分

  - 迭代部分
